What is Unix? and how does it work?
Unix is a multi-user, multi-tasking operating system (OS) that was developed in the late 1960s by Ken Thompson and Dennis Ritchie at Bell Labs. It's widely used in the tech industry and is known for its simplicity, flexibility, and reliability.

Here's a simplified overview of how Unix works:

**Key components:**

1. **Process**: A process is a sequence of commands that are executed by the operating system. Each process runs independently, and the system manager assigns resources (e.g., CPU, memory) to it.
2. **Kernel**: The kernel is the core of the Unix operating system. It provides low-level system services, such as memory management, I/O operations, and scheduling.
3. **Files**: Unix uses a file system to store data. Each file is a collection of files, and each file is a collection of lines of text.
4. **Directory**: A directory is a collection of files and subdirectories. It provides a hierarchical structure for storing and organizing files.
